Isekami Logo Design

category:

Logo Design, Java Programming

The goal of this project was to use Java to randomly generate shapes, and apply typography to it, to create a logo. The script output over 50 different shapes and eventually I paired one of them with the type treatment for "Isekami Paper Company" to form the Isekami Logo.
